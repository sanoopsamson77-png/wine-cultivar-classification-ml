# Wine Classification Project

This repository contains a complete data‑science project that predicts the
origin (cultivar) of Italian wines based on chemical analysis.  It uses
the Wine dataset from the UCI Machine Learning Repository.  The dataset
records 13 chemical attributes for 178 wine samples grown in the same region of
Italy but derived from three different cultivars.  The analysis determines
the quantities of constituents such as **alcohol**, **malic acid**, **proline** and
**color intensity** for each sample.  The task is to classify the wines into
three cultivars (classes) based on these measurements.

```
└── reports/
    ├── logistic_regression_report.csv  # Classification metrics for logistic regression

```

## Usage

1. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```

2. **Run the training script**

   Execute the training pipeline to perform EDA, train models and generate reports and figures:

   ```bash
   python src/train.py
   ```

   This will print performance metrics (accuracy and weighted F1 scores) for logistic regression,
   random forest and gradient boosting classifiers.  It will also create correlation heatmaps,
   feature importance charts and CSV classification reports inside the `figures/` and `reports/` directories.

3. **Explore the notebook**

   Open `notebooks/wine_classifier_analysis.ipynb` in Jupyter to interactively step through the analysis,
   experiment with different models, and visualise the data.

4. **Launch the Streamlit app**

   Run the Streamlit web application to interactively predict wine cultivars based on feature inputs:

   ```bash
   streamlit run app.py
   ```

   The app loads the Wine dataset, trains a random forest model and provides sliders for each chemical
   attribute.  Adjust the sliders in the sidebar to set feature values and view the predicted cultivar.
   A correlation heatmap is also displayed for reference (generated by running the training script).

## Results Summary

After running the training script on the entire dataset, the following performances were observed on the
held‑out test set:

| Model               | Accuracy | Weighted F1 Score |
|--------------------:|:-------:|:-----------------:|
| Logistic Regression | 0.9722  | 0.9720            |
| Random Forest       | 1.0000  | 1.0000            |
| Gradient Boosting   | 0.9444  | 0.9443            |

The random forest classifier achieved perfect accuracy, though the small dataset size means cross‑validation
should be used for robust evaluation.  The most important features across tree‑based models include
**color intensity**, **flavanoids**, **proline** and **alcohol**.
